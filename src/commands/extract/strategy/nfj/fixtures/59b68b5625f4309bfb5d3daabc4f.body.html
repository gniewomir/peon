Requirements description

We are looking for you if you have:

  • Advanced proficiency in TypeScript, including strong knowledge of types, generics, and strict null checks.
  • Experience building and shipping production apps with React Native (Expo), including Expo Router and NativeWind.
  • A solid understanding of React (Web), including component composition, hooks, and state management.
  • Experience working in a monorepo environment, particularly with pnpm workspaces and Turborepo.
  • Proficiency in styling with Tailwind CSS (web) and NativeWind (mobile), using design tokens and consistent color palettes.
  • Experience writing unit and integration tests with Jest (mobile) and Testing Library (web).
  • Familiarity with REST API integration using modern HTTP clients.

Nice to have:

  • Experience with react-native-web for cross-platform UI.
  • Familiarity with Rsbuild (Rspack) and @module-federation/rsbuild-plugin.
  • Experience working with CI/CD pipelines, automated testing, and code coverage enforcement.
  • Experience contributing to shared UI libraries and design systems.
  • Understanding of microfrontend architecture and experience with Module Federation.

Your traits:

  • Strong communication and collaboration skills in a distributed team.
  • Attention to detail and willingness to follow established conventions.
  • Proactive in learning new tools and frameworks as required by the project.

Offer description

Who we are?

  • A global team with high standards - we maintain Silicon Valley quality and vibe, even though we have spread far beyond.
  • We've made a real difference in the world since 2008! We deliver for Zoetis, Apple, Tesla, NATO, UNICEF, and many more.
  • 100-people-sized A-Team in Wrocław, with other offices in Poland, UK and Ireland.

Clear recruitment process:

  1. Introductory meeting - we are a close-knit team, and getting to know each other is our priority!
  2. Technical screening with React Native Engineer - after that stage, you won’t have any unanswered technical questions!
  3. A meeting with Director of Engineering - to get to know each other better and ensure we're aligned on values and expectations! 🤝

This is a B2B freelance position — we work on an hourly basis, and you’ll invoice us each month for the hours you’ve worked. The project is already ongoing, and we need someone immediately.

Your responsibilities

  1. Develop and maintain mobile features using React Native + Expo, ensuring cross-platform compatibility.
  2. Contribute to web microfrontends using React, following project structure and module federation patterns.
  3. Write and maintain unit and integration tests for all new features.
  4. Follow strict naming conventions and code organization as outlined in project documentation.
  5. Use shared workspace packages and avoid code duplication.
  6. Implement and maintain translations for all user-facing text.
